# Sample Codes for web3.js (Ethereum JavaScript API)  
## Environment  
```
Web3 + Ganache + Node.js + JavaScript + Solidity  
```
## Usage  
```
$ npm install  
$ cd contracts  
$ sh ganache.sh  
$ node ccd.js  
$ cd ..  
$ node web3-built-in.js  
$ node web3-contract.js  
```
